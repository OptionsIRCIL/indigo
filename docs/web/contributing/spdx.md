# SPDX Expressions

Contributors are highly encouraged to include SPDX license 
expressions at the top of their contributions.

For most of Indigo's codebase, the correct expression is:

```go
// SPDX-License-Identifier: AGPL-3.0-or-later
```

## Dual Licensing

If you wish to dual license your work under a more permissive license,
you MUST include an SPDX license expression as part of your contribution
that specifies it.

For example:

```go
// SPDX-License-Identifier: BlueOak-1.0.0 OR AGPL-3.0-or-later
```
